# Mis Gastos - Sistema de Gestión de Gastos Personales

Sistema moderno desarrollado con Laravel 11, PHP 8.3 y PostgreSQL 16.

## 🚀 Tecnologías

- **Backend**: Laravel 11.x
- **PHP**: 8.3
- **Base de Datos**: PostgreSQL 16
- **Servidor Web**: Nginx
- **Contenedores**: Docker & Docker Compose

## 📋 Requisitos

- Docker
- Docker Compose
- Git
